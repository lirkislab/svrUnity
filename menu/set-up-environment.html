Let's set up the environment!

1. Step

In this step we will visit unity page and download Unity Hub - a client, which procures all the unity versions

1.

Download Unity Hub: https://unity3d.com/get-unity/download

2.

Install Unity Hub

3.

After installation do a necessary steps like create account and such if required, set up licence for personal uses


2. Step

This step is dedicated to download concrete unity engine version

1.

Visit unity engine archive: https://unity3d.com/get-unity/download/archive

2.

Search for version 2019.4.14 and click on button Unity Hub as shown on picture bellow:

3.

You should see an pop up to open the downloaded file within unity hub - select this option

4.

When you successfully opened a downloaded file within UnityHub, similar window should show

and check:

- Microsoft Visual Studio Community 2019 <- otherwise you will code in different environment and you will need to set it up manually

- Android Build Support / iOS Build Support <- it depends on what mobile device you prefer to develop

- Windows build support / Mac Build Support <- it depends on what computer OS you prefer to develop

- And whatever else you would like to try by your own :)

5.

Wait for unity version to install ( it might take a while :) )


3. Step

In this step we will create and set up a project for exercises

1.

When unity version and modules installation is finished, create a project as shown on a picture below:

1 - Click new (select also correct version of unity (2019.4.14) if you have installed more of them)

2 - Make sure you are creating 3D project

3 - Give your project an awesome name

3 - Hit create

2.

Unity project should open, if it is requesting update, ignore it, ignore also your layout for now

3.

Download GoogleVR from this page: https://github.com/googlevr/gvr-unity-sdk/releases

To be more precise: Assets/GoogleVRForUnity_1.200.1.unitypackage

4.

Make sure, you can see "project" window in unity:

If not, navigate Window -> Windows -> Project

5.

Find your downloaded file (GoogleVRForUnity)

Drag and drop downloaded file as shown on picture bellow:

This window will pop-up:

This is called plugin/package, we can use its "prefabs" to create a VR game supported for mobile devices

You may get warnings about not used variables or that some function is absolete, but ignore it, its because Google stoped supporting VR mobile development for unity version 2017 and higher, but we want to you higher versions as older versions do not support stuff like new SDK's/NDK's/JDK's... yea and also a dark theme :)

6.

Navigate File -> Build Settings...

Window will pop up, u can choose android or iOS due to your preference, then select Switch Platform in right bottom corner

As our project is almost empty, this should take only a while. This process is necessary to be capable of building application on your desired mobile platfrom

7.

In a same window (File -> Build Settings...) click Player Settings... in a left bottom corner

You shall see Player, navigate:

Choose tab of your mobile device OS

Other Settings -> Graphics APIs - remove Vulkan by selecting it and then clicking "-" symbol

Other Settings -> Minimum API Level , for android its 19

Scroll down -> XR Settings -> check Virtual Reality Supported

You have empty list, click + and choose add Cardboard

4. Step

In this step we will make sure we have set-up our mobile device for development

1.

Make sure your phone is in development mode and has turned on "USB debuging", every phone is different so there is no way to describe process for all of them, here is an example for an adroind:

2.

Check, if you have downloaded NDK - otherwise you could not build apk directly into your phone

Navigate in unity: Edit -> Preferences...

Window will pop-up, select External Tools, scroll down you should see something like this:

This project is for android development and during unity version instalation in 2. Step , section 4 we selected android build support, therefore unity automatically set the paths

If your paths are empty, you could download SDK/JDK/NDK by your self and add the path

Or via Unity hub navigate: Unity hub -> Installs -> On your unity version: 3 vertical dots -> Add Modules (There you can download them too)

3.

After installation do a necessary steps like create account and such if required, set up licence for personal uses

5. Step

Try to build empty scene app on your phone (Yes, you will fall into the neverending abyss)

1.

Connect your phone to PC

2.

Navigate: File -> Build And Run

3.

Give your apk a name and save ;)

4.

Wait for magic to happen

5.

Voila , slap your phone on your forehead and you should be in VR!

(Not really, you would need a dioptric correction - headset)
